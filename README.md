## Module

```jsx
// python import랑 비슷한듯? : 모듈을 로드해서 객체 하나를 리턴

// math_tools.js
function add(a, b) {
  return a + b;
}
exports.add = add;

// main.js
// python import랑 비슷한듯? : 모듈을 로드해서 객체 하나를 리턴
let m = require('./math-tools.js');

console.log(m.add(1, 2));
```
